## â• `list + list` â†’ Concatenation

Python allows **adding two lists**, which means **combining** them:

```python

s_list = ['1', '2', '3']

combined = a_list + s_list
print(combined)
# Output: [1, 2, 3, '1', '2', '3']
```

- This **does not** add numbers â€” it **concatenates elements** from both.
    
- Result is a new list containing all elements from both lists.

## âœ–ï¸ `list * n` â†’ Repetition

Multiplying a list by a number **repeats** it that many times:

```python

# Output: ['1', '2', '3', '1', '2', '3', '1', '2', '3']
```

This is **not mathematical multiplication** â€” it repeats the sequence of elements.
