### ðŸ§© 3. Key Rules for Using `\frac{}`

| Part               | Syntax                          | Result               |
| ------------------ | ------------------------------- | -------------------- |
| Simple fraction    | `\frac{a}{b}`                   | $\Large \frac{a}{b}$ |
