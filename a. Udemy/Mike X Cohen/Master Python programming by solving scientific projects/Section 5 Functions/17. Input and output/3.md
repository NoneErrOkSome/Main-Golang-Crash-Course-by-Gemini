## ðŸ“ What Is `len()`?

- `len()` is a **built-in Python function** that returns the **number of items** in a container.
    
- Works with:
    
    - `list`
        
    - `tuple`
        
    - `str`
        
    - `numpy` arrays
        
    - And more...
        

---

## ðŸ“¦ Basic Example with Lists

```python

length = len(my_list)
print(length)  # Output: 3
```

- `len(my_list)` â†’ returns `3` because there are 3 elements.
    
- The result is stored in the variable `length`.

## âœ… Why It Matters

The output of `len()` is just a **regular integer** â€” you can use it in math:


> Here, you're dividing the sum by the length â€” both are function outputs.
