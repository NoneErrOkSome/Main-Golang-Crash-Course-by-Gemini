### üîç **Deep Dive: Part 2 ‚Äì Boolean Comparisons**

#### ‚úÖ Equality vs Assignment

In Python, you must **clearly distinguish** between:

- Assignment: $=$
    
- Comparison: $==$
    

**Assignment** puts a value into a variable:

```python
a = 4   # assigns the number 4 to variable a
```

**Comparison** checks if two values are equal:

```python
a == 4  # returns True if a is equal to 4
```

So in math terms:

- Assignment: $a = 4$
    
- Comparison: $a == 4$ (asks "Is $a$ equal to 4?")
    

#### üîÅ Example 1: Equal or Not

```python
print(4 == 4)     # True
print(4 == 4.5)   # False
```

- $4 == 4$ evaluates to $\text{True}$
    
- $4 == 4.5$ evaluates to $\text{False}$
    

#### üîº Inequality Operators

Python supports other comparison operators:

- $a > b$ ‚Üí greater than
    
- $a < b$ ‚Üí less than
    
- $a \geq b$ ‚Üí greater than or equal to ‚Üí `a >= b`
    
- $a \leq b$ ‚Üí less than or equal to ‚Üí `a <= b`
    
- $a \neq b$ ‚Üí not equal ‚Üí `a != b`
    

#### üîÅ Example 2: All Operators

```python
print(4 > 5)      # False
print(4 < 5)      # True
print(4 <= 4)     # True
print(4 >= 5)     # False
print(4 != 4.5)   # True
print(4 != 4)     # False
```

#### ‚ùó Capitalization Matters

```python
print(True)     # works
print(true)     # NameError
```

Only capital-$T$ `True` and capital-$F$ `False` are valid keywords.
