## ‚ú® **Basic Power Syntax**

In Python:

```python
3 ** 2
```

This means: $$ 3^2 = 9 $$

Python uses `**` for exponentiation (not `^`).

---

### üß™ Examples


- $2^3 = 8$
    
- $5^4 = 625$
    
- $10^0 = 1$

## üî¢ **Square Roots with Fractional Powers**

Python supports fractional exponents:

```python
9 ** (1/2)
```

$$ 9^{1/2} = \sqrt{9} = 3 $$

Note: the result is `3.0` because Python returns a float.

---

## ‚ö†Ô∏è **Order of Operations Mistake**

```python

```

Python interprets this as:

$$ (9^1) \div 2 = 9 \div 2 = 4.5 $$

‚úÖ To fix:

```python
9 ** (1/2)
```

$$ 9^{1/2} = 3 $$

## üìö **PEMDAS in Python**

Python follows standard math order:

- **Parentheses**
    
- **Exponents**
    
- **Multiplication / Division**
    
- **Addition / Subtraction**
    

So:

```python
a ** b / c
```

Means: $$ \frac{a^b}{c} $$
