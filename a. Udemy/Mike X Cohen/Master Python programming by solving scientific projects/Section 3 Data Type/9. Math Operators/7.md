## üß™ **Square Root Example**

You can compute square roots in Python using fractional exponents:

```python
9 ** (1/2)
```

Mathematically: $$ 9^{1/2} = \sqrt{9} = 3 $$

‚úÖ This is the correct way to compute a square root using exponentiation.

---

## ‚ö†Ô∏è **Common Mistake Without Parentheses**

If you accidentally write:

```python
9 ** 1 / 2
```

Python reads this as: $$ (9^1) \div 2 = 9 \div 2 = 4.5 $$

> ‚ö†Ô∏è This is not a square root. This is just dividing 9 by 2.

---

## ‚úÖ **Correct Form**

To ensure Python performs the correct operation first, use parentheses:

```python
9 ** (1/2)
```

$$ 9^{1/2} = 3 $$

## üß± **Python Operator Precedence (PEMDAS)**

Python follows this order:

1. **Parentheses** ‚Üí `()`
    
2. **Exponents** ‚Üí `**`
    
3. **Multiplication / Division** ‚Üí `*`, `/`
    
4. **Addition / Subtraction** ‚Üí `+`, `-`
    

So:

```python
2 + 3 * 4
```

Evaluates as: $$ 2 + (3 \times 4) = 14 $$

Use parentheses to change the order:

```python
(2 + 3) * 4
```

$$ (2 + 3) \times 4 = 20 $$
