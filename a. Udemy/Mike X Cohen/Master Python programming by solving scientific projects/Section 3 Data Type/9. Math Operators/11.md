## ğŸ’¡ **Operator Overloading Reminder**

When using `*`:

- With numbers: it performs **multiplication**
    
- With a string and an integer: it performs **repetition**
    

### Math Comparison:

$$ \text{"Mike"} \times 4 = \text{"MikeMikeMikeMike"} $$
