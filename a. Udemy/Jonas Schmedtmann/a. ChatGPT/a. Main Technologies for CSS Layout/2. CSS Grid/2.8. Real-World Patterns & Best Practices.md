### âœ… Favor `gap` over `margin`

### âœ… Use `minmax()` to balance fluid and fixed layout

### âœ… Use `auto-fit` over `auto-fill` when you want adaptive layout

### âœ… Combine Grid with Flexbox wisely

|Layout Role|Preferred Tool|
|---|---|
