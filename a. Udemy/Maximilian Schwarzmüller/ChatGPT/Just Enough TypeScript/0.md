## ðŸ“˜ 6. **Access Modifiers**

```ts
class Example {
  public a: string;   // accessible everywhere
  private b: number;  // accessible only inside class
  protected c: boolean; // accessible inside class & subclasses

  constructor(a: string, b: number, c: boolean) {
    this.a = a;
    this.b = b;
    this.c = c;
  }
}
```
